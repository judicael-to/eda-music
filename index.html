Analyse Exploratoire de Données Musicales

Un framework Python pour analyser les données Spotify et découvrir des insights musicaux

Voir sur GitHub

Fonctionnalités

Analyse Audio

Analysez les caractéristiques audio comme la danceability, l'energy et le tempo.

Analyse par Genre

Explorez les différences entre genres musicaux et leurs caractéristiques distinctives.

Visualisations

Générez des visualisations interactives et des rapports détaillés.

Installation

git clone https://github.com/votre-nom/eda-music.git
cd eda-music
python -m venv venv
source venv/bin/activate  # Sur Windows : venv\Scripts\activate
pip install -r requirements.txt

Exemple d'Utilisation

from src.data.load_data import load_spotify
from src.analysis.spotify_analysis import analyser_spotify_dataset

# Charger le dataset
df = load_spotify()

# Lancer l'analyse complète
analyser_spotify_dataset(df)
